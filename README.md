### How To Install
First, create Bundle directory
    mkdir ~/.vim/bundle

Then, clone Vundle
    git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

Finally, install plugins (in vim)
    :PluginInstall
